# Калькулятор на Python

Простое консольное приложение-калькулятор с базовыми арифметическими операциями.

## Функциональность

- Сложение
- Вычитание
- Умножение
- Деление
- Возведение в степень

## Установка

Клонируйте репозиторий:
```bash
git clone <url-репозитория>
cd pr13
```

Установите зависимости:
```bash
pip install -r requirements.txt
```

## Использование

Запустите калькулятор:
```bash
python calculator.py
```

## Тестирование

Запустите тесты:
```bash
pytest -v
```

Запустить конкретный тест:
```bash
pytest test_calculator.py -v
```

## CI/CD

Проект использует GitHub Actions для автоматического запуска тестов при каждом push и pull request.

### Что делает CI/CD пайплайн:

1. **Checkout code** - скачивает код из репозитория
2. **Set up Python** - устанавливает Python 3.11 с кешированием pip
3. **Display Python version** - отображает версию Python
4. **Upgrade pip** - обновляет менеджер пакетов pip
5. **Install dependencies** - устанавливает зависимости из requirements.txt
6. **Verify installation** - проверяет установленные пакеты
7. **Run tests with coverage** - запускает все тесты с отчетом о покрытии кода
8. **Test summary** - выводит итоговое сообщение

### Триггеры запуска:

- Push в ветки `master` или `main`
- Pull request в ветки `master` или `main`

### Просмотр результатов:

После push в GitHub перейдите во вкладку **Actions** вашего репозитория, чтобы увидеть результаты выполнения пайплайна.
